A&Y Appliances Chatbot Package

Files in this folder:
- server.js
- package.json
- .env.example
- squarespace-chat-widget.txt

What to do:
1. Create a Render account.
2. Create a new Web Service.
3. Upload these files using a GitHub repo, or copy them into Render later.
4. Add environment variables from .env.example.
5. After deployment, copy your Render URL.
6. Open squarespace-chat-widget.txt.
7. Replace https://YOUR-BACKEND-DOMAIN.com with your Render URL.
8. Paste the full widget code into Squarespace Code Injection, Footer.
